Exemplos de Tags Semânticas e Não Semânticas

Negrito / Destaque

Um exemplo de tag para deixar o texto em negrito com a tag <bold> (não semântico)

um exemplo para deixar o texto em destaque usando a tag <strong> (semântico)

Itálico / Ênfase

Um exemplo de texto em itálico usando a tag <i> (não semântico)

Agora um exemplo de texto com ênfase usando a tag <em> (semântico)

Texto Marcado

Podemos utilizar uma tag <mark> para criar um texto marcado, além disso, podemos utilizar um parâmetro style ou um seletor para alterar a cor do marcador.

Texto Grande e Texto Pequeno

Apesar de obsoleta, podemos utilizar a tag <big> para criar um texto grande.

E também podemos utilizar a tag <small> para criar um texto pequeno

Texto Excluído e Texto Inserido

Para criar um texto inserido, utilizamos a tag <ins>. Também existe a opção de usar a tag <u> (não semântica) para criar um texto sublinhado.

E para criar um texto excluído, podemos utilizar a tag <del>.

Texto Sobrescrito e Texto Subscrito

Podemos usar a tag <sup> para criar um texto sobrescrito como em x20+4.

Podemos também usar a tag <sub> para criar um texto subscrito, como em h2O
